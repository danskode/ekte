ekte

Agent-harnesset (Go TUI)

Det du installerer og kører. Al logik lever her — slash-kommandoer, sensor-loop, hooks, hukommelse. Provider-agnostisk, lokalt, bygget på AIDD.

Installation

Henter seneste release, verificerer checksum og installerer til ~/.local/bin:

curl -fsSL https://raw.githubusercontent.com/danskode/ekte/main/install.sh | sh

macOS & Linux · checksum-verificeret · Windows via WSL. Se forsiden for "inspicér først"- og go install-varianter.

Derefter:

cd dit-projekt && ekte

Scriptet lægger ~/.local/bin i din PATH. Åbn en ny terminal (eller source ~/.bashrc) bagefter — eller kør export PATH="$HOME/.local/bin:$PATH" for at bruge ekte i den nuværende shell.

Sådan indgår det i økosystemet

ekte er styringen. Det henter skills fra SKILLeton ved runtime, og kloner simple-minded som dit videnslager hvis du tilvælger en wiki. De to andre er rene data; ekte (med din valgte model) forbruger og vedligeholder dem.

AIDD-kerneflowet

  • /plan — kvalificér intentionen (ICE); /plan godkend udtrækker testbare succeskriterier.
  • /goal — autonom bygge-løkke: skriv kode → kør tjek-hook → inferentiel verifikation (sikkerhed + intent-conformance) → din accept.
  • /verify · /review — kør sensorerne ad hoc (også som ekte verify / ekte review til pre-push).

En /goal-kørsel er ikke færdig fordi koden kompilerer — den er færdig når den både består de tekniske tjek og beviseligt opfylder dine succeskriterier, og du har godkendt det. Hvorfor: AIDD →

Kommandoer i overblik

  • AIDD-flow: /plan · /goal · /verify · /review · /orchestrate
  • Skills: /skills library · show · install · bundle
  • Projekt/hooks: /init · /hook · /spec
  • Wiki/hukommelse: /wiki · /wiki-gem · /remember · /forresten
  • Sikkerhed: /dep · /sec-check · /security

Sikkerhedsmodel (kort)

  • Fil-tools er sandkasse-låst til projektmappen.
  • Tillid bestemmes af oprindelse, ikke kommando-streng; run_hook kræver bekræftelse.
  • Hemmeligheder redakteres før diffs sendes til en provider; sikkerheds-gates fejler lukket.
  • Persisteret kontekst (memory, ekte.md) bekræftes før skrivning.
